# Anime Skip Website

This project uses Nuxt 3, Dasiy UI, Pinia, and Apollo for it's main libraries.

## Required Tools

- NodeJS
- `pnpm`: `npm i -g pnpm`
- Docker

## Start Local Server

Install dependencies, and then run the `dev` command:

```bash
pnpm install
pnpm dev
```

### Configuration

```env
# Choose an API endpoint to use
VITE_API_BASE_URL=https://api.anime-skip.com
```
